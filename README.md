# London_Housing_Data_Analysis_Python_Pandas

## 📌 Project Overview

This project focuses on analyzing London housing and crime data using **Python, Pandas, Matplotlib, and Seaborn** in Jupyter Notebook. The objective was to perform data cleaning, preprocessing, exploratory data analysis (EDA), and statistical analysis to uncover trends in housing prices and crime rates across different areas of London.

Through this project, I applied core data analytics techniques such as handling missing values, date-time transformations, data filtering, grouping operations, and visualization to extract meaningful insights from a real-world dataset.

## 🎯 Key Objectives

* Clean and preprocess housing data for analysis.
* Analyze housing price trends across different years and regions.
* Examine crime statistics across London areas.
* Perform date-time analysis for time-based insights.
* Identify patterns using filtering, aggregation, and grouping techniques.
* Visualize missing values and data distributions.

## 📂 Dataset Information

The dataset contains housing and crime-related records from various areas of London, including:

* Date
* Area Name
* Average House Price
* Number of Crimes
* Housing Statistics
* Monthly and Yearly Records

Each row represents housing information for a specific area and time period.
